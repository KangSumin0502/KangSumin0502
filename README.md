# PPT 자동 생성기(PPTMaker)

키워드를 기반으로 간단한 PPT를 만들어 주는 python 프로그램입니다.


## 사용방법
1. 원하는 키워드를 아래와 같은 형식으로 텍스트 파일에 입력합니다. 

> [키워드1] [키워드2] [키워드3] ... [키워드n]

2. 텍스트 파일의 이름을 input.txt로 저장합니다.
3. 파일과 같은 위치에 프로그램을 저장합니다. 
4. 실행하면 result.pptx 파일과 summury.txt파일이 생성됩니다.

## 추천 사용자
- 당일 발표 자료를 두고 온 사람들
- 발표 자료를 만들어야 하는데 초안이 필요한 사람들

## 실행에 필요한 라이브러리
- python-pptx
- wikipedia API
- Pillow
